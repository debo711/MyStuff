To run the program:

-Unzip the the file and extract the solution folder

-You will need the a copy of Microsoft Visual Studio 2012 or newer as it is
 incompatible with the older version, Microsoft Visual Studio 2010, and it is 
 written in C#

-Open the solution folder and find the .sln file

-Double click the .sln file and if the correct version of Microsoft Visual Studio
 is installed, the program solution should open to view code.

-To run the program, click the Start button with the Debug option selected
